# App Preview Video Script (20-30s)

Date: 2026-02-18

## Goal
Show fast value: scan -> extract -> edit -> export.

## Timeline

0s-3s
- Shot: App open + scan button tap.
- On-screen text: "Scan docs offline"

3s-8s
- Shot: capture 2-3 pages in one session.
- On-screen text: "Multi-page in one flow"

8s-13s
- Shot: extracted text appears.
- On-screen text: "Automatic text detection"

13s-18s
- Shot: user edits amount/date/reference.
- On-screen text: "You stay in control"

18s-23s
- Shot: category/domain auto classification.
- On-screen text: "Invoice, receipt, contract"

23s-28s
- Shot: export menu PDF/Excel/TXT/JSON.
- On-screen text: "Export where you work"

28s-30s
- Shot: logo + CTA.
- On-screen text: "No cloud. No login. 100% offline"

## Production specs
- Format: MP4 (H.264)
- FPS: 30
- Resolution: vertical device-native
- Audio: optional, captions mandatory

## Store specs reminder
- App Store: jusqu'a 3 previews par device family, 15-30s recommande
- Google Play: promo video YouTube optionnelle (fortement conseillee)
- Toujours eviter toute donnee personnelle visible dans la video
