# App Store Screenshots Kit (benefits-first)

Date: 2026-02-18

## Strategy
Show user outcomes, not technical internals.
Each screenshot should answer: "What do I gain in 3 seconds?"

## Recommended sequence (6 screenshots)

1. Headline: "Scanne 1 ou 12 pages en quelques secondes"
   - Visual: scan flow with multi-page queue.
   - Benefit note: save time on repeated docs.

2. Headline: "Tes documents restent 100% sur ton telephone"
   - Visual: privacy-friendly screen + offline badge.
   - Benefit note: no cloud dependency.

3. Headline: "Facture, recu ou contrat: classement auto"
   - Visual: list view showing mixed categories.
   - Benefit note: faster archive and retrieval.

4. Headline: "Corrige le texte et les donnees avant sauvegarde"
   - Visual: editable text and extracted fields.
   - Benefit note: reduce data-entry mistakes.

5. Headline: "Retrouve un document en 2 secondes"
   - Visual: search bar with result cards.
   - Benefit note: instant access during admin tasks.

6. Headline: "Exporte en PDF, Excel, TXT ou JSON"
   - Visual: export panel with known file icons.
   - Benefit note: ready for accountant/team sharing.

## Text rules
- Keep headline <= 8 words.
- Use high contrast overlays.
- One message per screenshot.
- Avoid tiny body text.
